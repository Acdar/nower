Mower 是为长期运行设计的、开源的明日方舟脚本。
Mower 致力于为用户提供最佳服务，全天候 24\*7 托管支持，旨在让用户彻底解放双手，忘记《明日方舟》，并且帮助用户获取游戏内最多资源。

## 功能介绍

- 基建：跑单、按心情动态换班；
- 森空岛：签到、仓库读取；
- 日常：公招、邮件、线索、清理智；
- 大型任务：生息演算、隐秘战线；
- 签到：五周年月卡、限定池每日一抽、矿区、孤星领箱子、端午签到……
- 调用 maa：肉鸽、保全。
